# ЗАДАЧА:

Компания активно растет и хочет перейти к динамическому ценообразованию, чтобы:

    * лучше балансировать спрос и предложение
    * увеличивать выручку
    * повышать доступность самокатов в нужных местах

Сейчас у компании есть исторические данные о поездках, пользователях, погоде и некоторых акциях, но нет четкого понимания, как именно устроен спрос, и какие факторы на него влияют.

Необходимо провести исследование и ответить на вопрос: как устроен спрос на самокаты и какие факторы нужно учитывать, чтобы внедрить динамическое ценообразование?


Исследовательский анализ при этом разделен на несколько частей, каждая с анализом одного из факторов - время, цена, загруженность по локациям и погода, далее сделан вывод по взаимодействию каждого фактора со спросом.
